Обновление загрузчика Windows при развертывании рядом с Linux

01.10.2024Меньше 1 минутыWindowsboot

При установке Windows рядом с Linux после перезагрузки вылетает ошибка:

0xc0000225 Error on Windows Installation to GPT (UEFI)

Для её исправления необходимо исправить загрузчик в загрузочном разделе. Если у вас есть доступ к другому компьютеру, извлеките основной диск компьютера и найдите способ смонтировать его в системе (внутренне, внешне, как угодно). Либо запустить установку windows с флешки и выбрать восстановление системы и запустить командную строку Shift +F10.

Затем выполните следующие действия:

Откройте командную строку от имени администратора и введите

diskpart

Введите команду для показа списка дисков, чтобы посмотреть к какому относится наш диск

list disk

Введите следующую команду, где # номер диска

select disk #

Введите команду, и найдите загрузочный раздел, обычно он 100-1000 Мб

list partition

Выберите этот раздел, где # номер раздела

select partition #

Назначьте букву для раздела, например Z, предполагая что она не используется

assign letter=z

Выйдите из diskpart используя сочетание CTRL +C

Введите следующую команду, подразумевается что windows находится на диске C:

BCDBoot C:\Windows /S Z: /F UEFI

Это должно исправить загрузчик. Если вы можете попасть в среду восстановления на компьютере, то это также должно сработать.
